# AR-GARCH VaR Backtesting

This project implements an AR-GARCH framework to model financial return volatility and estimate Value-at-Risk (VaR).

## Features
- AR model for return dynamics
- GARCH model for volatility clustering
- Parametric VaR estimation
- Rolling window forecasting
- Kupiec POF backtesting

## Libraries Used
- pandas
- numpy
- statsmodels
- arch
- matplotlib

## Results
The model captures volatility clustering and evaluates risk forecasting performance using statistical backtests.
